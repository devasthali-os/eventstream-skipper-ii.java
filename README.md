Eventstream with Google PubSub/ RabbitMQ
----------------------------------------

- Java pipeline to stream events to Google Pub sub/ RabbitMQ using Spring, PubSub SDK
- JMeter perf is included in 

eventstream
-----------------------

```
              |----------------------------------
publisher ~>  | 1 | 2 | 3 | 4 | 5 | ........| n |    ~>  subscription ~> Subscriber 
              |----------------------------------     
                        stream (topic)
```
